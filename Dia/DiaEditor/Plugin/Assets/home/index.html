Cluiche Editor

No project is loaded. Editor panels registered by plugins will appear here once a project is opened.

Open a project

Launch with a project path on the command line:

CluicheEditor.exe Data/test.cluicheproj

Shortcuts

  • Ctrl+Z — Undo
  • Ctrl+Y / Ctrl+Shift+Z — Redo
  • Ctrl+Shift+P — Command Palette
This panel is always registered by EditorView so the docking layout is never empty.
